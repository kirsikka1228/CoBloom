# AI 知识库与智能问答平台需求文档 V2.0

## 1. 项目概述

### 1.1 项目名称

AI 原生个人知识库与智能问答平台

### 1.2 项目定位

本项目的定位不是简单的 Markdown 管理系统，而是构建一个 AI
驱动的个人知识库系统。

系统参考 AI Knowledge Base 思想： - 用户输入原始知识资料； - AI
自动理解、整理和结构化知识； - 构建知识关系网络； -
基于知识库进行智能问答； - 用户探索过程产生的新知识反哺知识库。

核心理念：

> Markdown 是知识输入形式，而不是最终知识形态。

------------------------------------------------------------------------

# 2. 项目目标

系统实现：

1.  Markdown 知识采集与管理；
2.  AI 自动知识编译（Knowledge Compiler）；
3.  自动摘要、关键词、概念和实体提取；
4.  文档切片与向量索引；
5.  基于知识库的 RAG 智能问答；
6.  轻量级知识图谱构建；
7.  知识关联推荐；
8.  AI 辅助知识整理和维护。

------------------------------------------------------------------------

# 4. 功能需求

## 4.1 知识输入模块

支持：

-   Markdown 在线编辑；
-   Markdown 文件上传；
-   原始文档保存；
-   文档状态管理。

文档状态：

-   RAW：原始资料
-   PROCESSING：AI处理中
-   COMPLETED：知识构建完成

------------------------------------------------------------------------

## 4.2 AI Knowledge Compiler

系统核心模块。

用户保存文档后，AI 自动执行：

    Markdown
     ↓
    文本解析
     ↓
    内容清洗
     ↓
    智能分块
     ↓
    Embedding
     ↓
    摘要生成
     ↓
    关键词提取
     ↓
    概念提取
     ↓
    实体提取
     ↓
    知识关系生成

生成内容：

-   Summary
-   Keywords
-   Concepts
-   Entities
-   Related Knowledge

------------------------------------------------------------------------

## 4.3 文档分块与向量索引

功能：

1.  Markdown 标题结构解析；
2.  按章节和语义切片；
3.  保存 Chunk；
4.  生成 Embedding；
5.  建立向量检索能力。

Chunk 数据：

-   noteId
-   content
-   position
-   embedding

------------------------------------------------------------------------

## 4.4 AI 智能问答

采用增强 RAG 架构：

    用户问题
    
     ↓
    
    Query Embedding
    
     ↓
    
    向量检索
    
     ↓
    
    关键词检索
    
     ↓
    
    知识关系扩展
    
     ↓
    
    LLM生成回答
    
     ↓
    
    回答 + 引用来源

要求：

-   禁止脱离知识库直接回答；
-   返回引用文档；
-   保存问答历史；
-   支持基于知识节点继续追问。

------------------------------------------------------------------------

## 4.5 知识图谱模块

**目标：**展示知识结构，而不是数据库关系。

**节点类型：**

  类型       说明
---------- --------------
  Note       原始笔记
  Concept    知识概念
  Entity     知识实体
  Question   用户关注问题

**关系：**

  关系           说明
-------------- ------
  contains       包含
  related        相关
  derived_from   来源
  similar_to     相似

**前端：**

使用 ECharts Graph。

支持：

-   缩放；
-   拖拽；
-   节点点击；
-   查看详情；
-   局部知识图谱。

------------------------------------------------------------------------

## 4.6 关联推荐

推荐依据：

    推荐分数=
    
    向量相似度
    +
    关键词匹配
    +
    知识图谱关系
    +
    标签关系

展示：

-   相关笔记；
-   相关概念；
-   推荐阅读路径。

------------------------------------------------------------------------

## 4.7 AI知识维护

新增：AI Health Check。

功能：

-   检查重复知识；
-   发现缺失关系；
-   推荐新知识节点；
-   检测知识冲突。

------------------------------------------------------------------------

而是：

"AI驱动的个人知识管理与知识探索系统"。

核心技术亮点：

-   Knowledge Compiler
-   RAG
-   Embedding Retrieval
-   Knowledge Graph
-   AI Knowledge Evolution
